Consignas para el integrador etapa de acompañamiento:

Para esta instancia se deberán crear un proyecto de acceso publico en gitlab (con su cuenta), el cual tenga 2 ramas. Compartiendo en link por mail
En la medida que vayan subiendo contenido al repositorio deberán asi mismo enviar una notificación al docente, ya sea por mail o configurando el repo para que sea el, el que notifique.

Recuerden darme permisos dentro de su proyecto para que pueda bajarlo y evaluarlo)

Requerimientos:

1er Parte

1.- Creación de un docker con Apache - mysql y PHP (versión standard)
con un archivo Readme, que contenga las instrucciones de como levantar el mismo, paso a paso.-

2.- Recuerden que solo debe estar en el repositorio lo fundamental del mismo, sin arhivos innecesarios, recuerden utilizar .gitignore (no traer contenido de mysql y demas) . El repositorio en el mejor de los casos solo contendra archivos de textos y no binarios. No debe pesar mas de 2/5Mb

2da Parte

3.- Se deberá crear un script .sh que se pueda correr una vez que este el contenedor arriba que genere la BD , tablas , vistas , triggers y demas.-

4.- Los datos correspondientes a las tablas se encuentran mencionados a continuación, en donde deberán descargar las mismas y realizar una BD que se llame MapaCultural, que contengan las diferentes ofertas que se mencionan para descargar, como salas de cines, teatro, etc. Los datos que bajaran no esta relacionados y usted deberá llevar a cabo la tarea de la normalización de toda la base, es muy importante que así lo pueda desarrollar para poder llevar adelante esta etapa. Recuerde que pueden resultar tablas pivotables, bridge.
